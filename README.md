# Workout Management System

A project for managing exercises, focused on learning and practicing web architecture.

## 🛠 Current Features
- CRUD operations: List, create, update, and delete exercises.
- Frontend: Vanilla JavaScript (ES6 Modules).
- Backend: Spring Boot REST API.

## 🚀 Future Steps
- Implement workout plan (fichas) system.
- Improve error handling and data validation.
- Add search functionality.
